About OptiFly

OptiFly is an Airline Route Optimizer project designed to help users find the most efficient air routes based on cost, time, or a balance of both. It also provides administration and client management features for airline operations.

Features

  • Route Optimization: Find optimized air routes between airports based on user-selected priorities.
  • Flight Administration: Add and manage flight details, including costs for different classes.
  • Client Management: Add, update, delete, and view client information.
  • Modern UI: Clean, responsive, and easy-to-navigate frontend.

Project Structure

  • frontend/ - All user and admin HTML files
    • index.html - Main user interface (route optimizer)
    • admin/client.html - Flight administration
    • client/ - Client management (add, update, delete, view)
  • Administrator/ - Java backend for airport and flight data
  • client/ - Java backend for graph and path optimization

How to Use

  • Open frontend/index.html in your browser for the main route optimizer.
  • Use the navigation buttons to access client management and flight administration.
  • For backend features, compile and run the Java files as described in the project documentation.
